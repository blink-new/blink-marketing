# Reference Examples — Real Launch Posts

These are real posts that performed. Study them before writing. Pattern-match against them. The best ones are annotated with what makes them work.

---

## SAME.DEV — X — 7.2M VIEWS
**The highest performer in this collection. 3 lines.**

```
Introducing Same.dev

Clone any website with pixel perfect accuracy

One-shots Nike, Apple TV, Minecraft, and more!
```

**Why it works:**
- One thing only: clone any website
- "Pixel perfect accuracy" is a specific, verifiable claim
- Three named examples (Nike, Apple TV, Minecraft) make it concrete and impressive immediately
- No giveaway mechanic — proves that a strong enough claim doesn't need one
- Shortest post, highest views. The lesson: if your claim is strong enough, get out of the way.

## OPENWORK — X — 987K VIEWS

```
Today we are launching @openwork_ai, an open-source (MIT-licensed) computer-use agent that's fast, cheap, and more secure.

@openwork_ai is the result of a short two-day hackathon our team decided to hack, which brings together some of our favorite open source AI modules into one powerful agent, to allow you to:

1. Bring your own model/API key (any provider and model supported by @opencode is supported by Openwork)

2. ~4x faster than Claude for Chrome/Cowork, and much more token-efficient, powered by dev-browser by @sawyerhood (legend)

3. More secure - contrary to Claude for Chrom/Cowork, does not leverage the main browser instance where you are logged into all services already. You login only to the services you need. This significantly reduces the risk of data loss in case of prompt injections, to which computer-use agents are highly exposed.

4. Free and 100% open-source!

You can download the DMG (macOS only for now) or fork the github repo via the link in bio (@openwork_ai).

Let us know what you think (or better, send a pull request)!
```

**Why it works:**
- "fast, cheap, and more secure" — three specific competitive claims upfront
- "two-day hackathon" story makes it feel real and founder-led
- "~4x faster" is a specific, comparative benchmark — not just "faster"
- Tagging collaborators (@sawyerhood) is authentic and drives retweets from their network
- No giveaway mechanic — performs on the strength of the technical credibility alone
- Note: this is the longest post in the set and still performs well, but underperforms relative to length vs. shorter posts like Same.dev
